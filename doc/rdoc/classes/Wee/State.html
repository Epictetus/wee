Class Wee::State
In: lib/wee/state.rb
Parent: Object

This class is for backtracking the state of components (or decorations/presenters). Components that want an undo-facility to be implemented (triggered for example by a browsers back-button), have to overwrite the Component#backtrack method. Class Wee::State simply represents a collection of objects from which snapshots were taken via methods take_snapshot.

    def restore
      @objects.each_value {|s| s.object.restore_snapshot(s.snapshot) }
      @objects_ivars.each_value {|s|
        s.ivars.each {|k,v| s.object.instance_variable_set(k, v) }
      }
